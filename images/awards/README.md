# Awards & Certifications

Add your awards and certificates here. They will be displayed in the Awards section.

## Recommended:
- Professional certifications
- Awards and recognitions
- Achievement certificates
- Course completion certificates

## File Format:
- JPEG or PNG
- Recommended size: 600x800px (portrait orientation)
- Ensure good quality for readability

## How to Use:
1. Add your certificates/awards to this folder
2. Update the HTML in `index.html` to reference your images:
   ```html
   <div class="award-card">
       <img src="images/awards/your-certificate.jpg" alt="Certification">
   </div>
   ```
